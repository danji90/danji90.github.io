# Online portfolio

Online portfolio for Daniel Marsh-Hunn. This online portfolio is bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Applied technologies

- [NodeJs](https://nodejs.org/en)
- [ReactJS](https://reactjs.org/)
- [React-Redux](https://react-redux.js.org/)
- [Material-UI](https://mui.com/)
- [Openlayers](https://openlayers.org/)
- [react-spatial](https://react-spatial.geops.io/)
- [mobility-toolbox-js](https://mobility-toolbox-js.geops.io/)
- [react-tooltip](https://wwayne.github.io/react-tooltip/)
- [react-image-gallery](https://github.com/xiaolin/react-image-gallery')
- [react-github-calendar](https://grubersjoe.github.io/react-github-calendar/)
